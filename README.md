# 🏥 Web-Based Intelligent Medical Triage & Patient Prioritization System

**An AI-integrated Health-Tech solution designed to optimize Emergency Department (ED) workflows using Generative AI and Full-Stack Web Development.**

---

## 📝 Abstract
Emergency Department overcrowding often delays life-saving care. This project introduces a **Web-Based Medical Triage System** that automates patient prioritization using **Node.js, SQL, and the Google Gemini API**. By applying Generative AI to analyze natural language symptom descriptions, the system provides instant, objective urgency scores. This decision-support tool reduces assessment latency and human error, ensuring a "most-urgent, first-served" workflow to optimize hospital resources and improve patient survival rates.

---

## 🚀 Key Features
- **AI-Driven Decision Support:** Utilizes Google Gemini API for NLP-based symptom analysis.
- **ESI Standard Prioritization:** Accurate clinical prioritization based on Emergency Severity Index standards.
- **Real-Time Risk Assessment:** Instant urgency classification for critical cases.
- **Secure Data Persistence:** Robust SQL backend for medical accountability and logging.
- **Responsive Dashboard:** Real-time visibility for hospital staff to manage bed and staff allocation.

## 🏗️ Technical Stack

- **Backend:** Node.js, Express.js
- **Database:** SQL (Structured Database Management System)
- **AI Integration:** Google Gemini Pro API (Prompt Engineering)

---

## ⚠️ Disadvantages & Constraints
- **Internet Dependency:** Requires stable connectivity for Gemini API access.
- **No Physical Exam:** Cannot replace manual checks like palpation or auscultation.
- **Decision Support Only:** Final medical decisions must always be validated by a professional.

## 🏁 Conclusion
By shifting from a "first-come" basis to an objective "most-urgent, first-served" workflow, this system significantly enhances the efficiency of Emergency Departments. As AI technology continues to advance, such systems will become essential in ensuring the right patient receives the right care at the right time.
